Random Number API

Introduction

This is a sample application that demonstrates what Vex can do. It implements a simple API with an endpoint that returns pseudorandom numbers in the specified range.

Instructions

  1. Navigating to the root returns this homepage.

  2. /:generator returns a uniformly distributed pseudorandom 32-bit integer generated by the specified generator.

    The list of supported generators is specified in the generators section below.

    Examples:

    • /rng
    • /mt
    • /sm

  3. /:generator?min=x&max=y&float=true returns a uniformly distributed pseudorandom number in the specified range. A floating point number is returned if float is set to true, otherwise an integer is returned.

    All supported query parameters are listed in the query parameters section below.

    Examples:

    • /rng?min=1&max=10
    • /mt?min=-1&max=+1&float=true
    • /sm?max=100

Generators

Generator Name Identifier Description
Default rng The default RNG used by V. Generally WyRand.
Mersenne Twister mt Mersenne Prime RNG implementation.
SplitMix64 sp Java 8's SplittableRandom generator implementation.

Query Parameters

Parameter Default value Description
min 0 The minimum value that can be returned. Inclusive limit.
max 232 for integers and 1.0 for floats. The maximum value below which values can be returned. Exclusive limit.
float false Set this to true if floating point numbers are required. Returns integers otherwise.
